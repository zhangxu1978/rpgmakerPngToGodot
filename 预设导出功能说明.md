# 预设导出功能说明

## 功能概述
在切图预览界面中新增了根据预设导出图片的功能，支持9tile、16tile、48tile三种预设模式。

## 使用方法

### 1. 准备工作
1. 打开图片并进行切图操作
2. 进入切图预览界面
3. 确认切图或查看切图结果

### 2. 预设导出
1. 在切图预览界面找到"预设导出"区域
2. 从下拉菜单中选择预设类型：
   - **9tile**: 3×3网格，共9个组合图块
   - **16tile**: 4×4网格，共16个组合图块  
   - **48tile**: 12×4网格，共48个组合图块
3. 点击"导出预设图片"按钮
4. 选择保存位置和文件名
5. 系统将自动生成组合后的图片

## 预设文件格式

预设文件为JSON格式，包含以下字段：

```json
{
  "cols": 12,        // 输出图片的列数
  "rows": 4,         // 输出图片的行数
  "cellData": [      // 每个大方块的组合数据
    [9, 12, 13, 16], // 第一个大方块：由第8,11,12,15个切片组成
    [9, 10, 13, 8],  // 第二个大方块：由第8,9,12,7个切片组成
    // ... 更多组合数据
  ]
}
```

### cellData说明
- 每个数组包含4个数字，代表一个2×2的大方块
- 数字表示切片的索引（从1开始计数）
- 四个数字按左上、右上、左下、右下的顺序排列
- 例如：[9, 12, 13, 16] 表示将第8,11,12,15个切片组合成一个大方块

## 输出结果
- 导出的图片将按照预设的网格布局排列
- 每个网格单元由4个小切片组合成一个大方块
- 最终图片尺寸 = 列数 × (切片宽度 × 2) × 行数 × (切片高度 × 2)

## 注意事项
1. 确保已完成切图操作，有可用的切片数据
2. 预设文件必须存在于应用根目录
3. 切片索引超出范围时会自动跳过并在控制台显示警告
4. 建议切图尺寸为48×48像素以获得最佳效果
